# Images des HTTP Status Cats

Place tes images de chats ici avec les noms suivants :

- `200.webp` - Chat content/heureux
- `401.webp` - Chat qui demande "Qui es-tu?" / Chat policier qui demande les papiers
- `403.webp` - Chat qui dit non/qui bloque (bouncer/garde)
- `404.webp` - Chat perdu/confus
- `418.webp` - Chat dans une théière (ou avec une théière)
- `422.webp` - Chat qui examine/valide des formulaires avec erreurs
- `500.webp` - Chat paniqué/stressé en plein chaos
- `204.webp` - Chat vide/minimaliste/zen

Format : WEBP, environ 800x600px ou 16:9
